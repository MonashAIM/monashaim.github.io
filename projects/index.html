Ischemic Stroke Detection

Our team's first project was to develop a machine learning model to detect ischemic strokes. This project used the ISLES22 dataset to create a basic model, using HighResnet. While this project wasn't able to quite get to the same accuracy other researchers achieved, it was a big step for the team. It allowed us to develop a research and project methodology which we have applied to our current projects. Future team research into this area could involve finding ways to improve upon the AI model, and a way to deploy the application.
